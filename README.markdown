Hydra daemon [![BS][BSI]][BSURL] [![CS][CSI]][CSURL]
====================================================

Proof-of-concept [libuv] daemon for running other programs.

**Tested using [Travis CI] with Clang 3.3 and GCC 4.6.3.**

Check out the [Github repo] for the source code.
